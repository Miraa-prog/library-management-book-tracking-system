# 📚 Library Management & Book Tracking System

## 📌 Overview
This project is a database system designed to manage library operations including books, users, and borrowing tracking.  
It demonstrates real-world database design, ER modeling, and SQL implementation.

---

## 🎯 Problem Statement
Libraries need an efficient system to manage books, users, and borrowing records while ensuring data consistency and easy retrieval.

---

## 💡 Solution
A relational database system was designed using MySQL to:
- Store and manage books and users
- Track borrowing and returning processes
- Maintain relationships between entities
- Ensure data integrity and normalization

---

## 🛠 Technologies Used

- SQL
- draw.io (ER Diagram & Schema Design)
